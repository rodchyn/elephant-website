Project Goal.

This is CoffeScript inspired sugar syntax project for PHP programming language. Author of this syntax addidion is Yura Rodchyn PHP developer from Ukraine. The motivation was the hatred of the "$, -> and ;" and main goal of project is next sugar syntax:

$a = 'b';

class Echo {
    private $var;
    public function __construct($var)
    {
        $this->var = $var;
    }
}

converted from:

a = 'b'

class Echo
    construct -> (@var)

Also with few missed functional addition like:

a = 'bcdef'.length
$a = bm_strlen('bcdef');
